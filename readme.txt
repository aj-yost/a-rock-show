# A Rock Show Website Package

This is a complete static website package.

## Files
- index.html — homepage
- calendar.html — show calendar
- booking.html — booking page
- about.html — about page
- styles.css — shared styling
- images/ — logo and band photos

## Replacing Images
I could not access the original poster source assets from the available File Library search results, so this package includes placeholder images.

To use your real assets, replace these files while keeping the same names:

- images/logo.svg
- images/band-photo.svg
- images/member-1.svg
- images/member-2.svg
- images/member-3.svg
- images/member-4.svg

You can also use .jpg or .png files, but if you change the file extension, update the image paths in the HTML.

## Recommended Hosting
Use Netlify for easiest deployment:
1. Go to netlify.com
2. Drag this whole folder onto Netlify
3. Netlify will publish the site
4. Connect your custom domain later

## Booking Form
The booking form currently uses mailto:yourbandemail@example.com.
Replace that email in booking.html with your real email.
For a more reliable form, use Netlify Forms, Formspree, or another form provider.
